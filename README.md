# Analise Estatistica-BMW-2010-2024
Análise estatística da BMW entre 2010 e 2024 com estudo de modelos, regiões, combustíveis, quilometragem e transmissões.
Projeto de análise estatística desenvolvido com dados da BMW entre 2010 e 2024. O estudo investiga o comportamento dos modelos ao longo do tempo, a expansão dos SUVs, a adoção de tecnologias híbridas e elétricas, a distribuição global das vendas, os tipos de combustível, a quilometragem média e os padrões de transmissão, identificando tendências e transformações do mercado automotivo premium.
